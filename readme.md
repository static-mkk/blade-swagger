# Blade-Swagger

---

    该仓库是模仿springfox（整合springmvc和swagger)来进行整合Blade和swagger的仓库. 


此时：2018/8
模仿的是第一版：[swagger-springmvc-0.1.0][1]


***由于网上关于整合swagger的中文文章很少，所以我选择了直接 模仿springfox.
当然，直接去读swagger的英文文档也是可以，但是相比较之下，还是前者比较省时省力.***
